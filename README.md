# src_viewer
# 简介
## 描述
```
1.本地资源预览的前台部分，用vue2脚手架编写

2.开发环境下编码为UTF-8,打包使用时编码为GB18030

3.完整资源预览GUI项目github地址:    
    https://github.com/ShenWei-HopeNoBug/srcViewerGUI
```
## 原理
```
1.桌面GUI搜索和条件筛选出资源信息，将资源信息作为全局变量写入
  imgPath.js/videoPath.js/musicPath.js静态资源文件中供前端预览
  
可配置筛选条件：
    搜索文件夹
    忽略文件夹
    筛选扩展名
    忽略扩展名
    文件大小范围(MB)
    超出大小范围的搜索文件是否占位显示

2.GUI也可通过文件操作修改option.js静态资源文件中有关显示的全局变量修改前端显示
  
可配置显示设置：
    起始页：默认为第 1 页
    单页行数：默认为 3 行
    打开页面切换到路由：默认为/image
```
# 页面路由
## 路由结构
```
-->/image
-->/video
-->/music

打开页面后默认先切到哪个路由取决于桌面GUI如何配置的
路由切换后会直接跳到第一页
```
## 路由描述
### /image
```
1.用于预览图片，选用了viewerjs第三方图片预览插件实现了单多图预览模式切换

2.图片加载模式为顺序加载，在顺序加载触发前会先加载一张loading图片

3.图片加载出错或超出搜索大小范围会显示对应提示图片

4.鼠标悬浮显示该图片在本地的绝对路径，方便查找本地资源位置
```
### /video
```
1.用于预览视频，选用了video.js第三方视频插件实现了基本视频播放功能

2.限制同一页只能播放一个视频

3.视频加载出错或超出搜索大小范围会显示对应提示图片，且不再可控

4.鼠标在视频名位置悬浮会显示该视频在本地的绝对路径，方便查找本地资源位置
```
### /music
```
1.用于预览音频，解析歌曲得到的封面会覆盖默认的唱片外观，点击播放唱片开始旋转

2.超出搜索大小范围唱片封面会显示提示图片，且不再可控

3.双击歌名区域会向上弹出该音频在本地的绝对路径，方便查找本地资源位置
```
